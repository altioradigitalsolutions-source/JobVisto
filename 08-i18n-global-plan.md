# I18n Global Plan

## Vision

El producto debe nacer preparado para uso mundial. No debe sentirse como una herramienta local de un solo pais.

## Idiomas iniciales

1. Espanol.
2. Ingles.
3. Hebreo.
4. Ruso.

## Orden de trabajo

Primero se define todo en espanol:
- conceptos,
- pantallas,
- botones,
- mensajes,
- reportes,
- emails/WhatsApp,
- estados,
- precios base.

Despues se traduce a:
- ingles,
- hebreo,
- ruso.

## Reglas tecnicas futuras

- Usar i18n desde el inicio.
- No quemar textos directamente en componentes.
- Crear archivos por idioma.
- Guardar idioma preferido por usuario/organizacion.
- Permitir cambiar idioma desde configuracion.
- Soportar moneda, formato de fecha y zona horaria por organizacion.
- Soportar RTL para hebreo.

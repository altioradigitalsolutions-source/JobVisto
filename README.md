# Cleaning SaaS Project

Base de producto para un SaaS de gestion de limpieza orientado a:
- Cleaners independientes.
- Empresas pequenas y medianas de limpieza.

El objetivo no es procesar pagos entre clientes, empresas y limpiadores. El sistema organiza, registra, calcula, estima y deja comprobantes digitales. La monetizacion principal es membresia mensual.

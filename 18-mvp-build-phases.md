# MVP Build Phases

## Principio

Construir primero el flujo que demuestra valor real:

Crear trabajo -> cleaner llega -> fotos/checklist -> termina -> cliente firma -> sistema calcula -> reporte.

Despues se agrega pago recibido y empresa avanzada.

## Fase 0: Preparacion

- Confirmar nombre temporal o final.
- Elegir stack.
- Crear app base.
- Configurar auth.
- Configurar base de datos.
- Configurar i18n.
- Crear layout base.

## Fase 1: Auth y onboarding

Incluye:
- registro con email/contrasena,
- verificacion email,
- login,
- login con Google,
- elegir tipo de cuenta,
- crear organizacion,
- idioma,
- pais,
- moneda,
- zona horaria.

Resultado:
Usuario puede entrar y tener espacio propio.

## Fase 2: Independiente basico

Incluye:
- dashboard independiente,
- clientes,
- crear cliente,
- crear trabajo,
- calendario mensual propio,
- navegacion de meses,
- dias vacios visibles,
- detalle expandible de servicio,
- link a Google Maps desde direccion,
- detalle del trabajo.

Resultado:
Un cleaner independiente puede organizar su agenda y clientes.

## Fase 3: Ejecucion movil del trabajo

Incluye:
- vista movil del trabajo,
- boton Llegue,
- GPS al llegar,
- boton Termine,
- calculo de duracion,
- checklist,
- fotos,
- notas.

Resultado:
El cleaner puede operar desde telefono.

## Fase 4: Cliente y firma

Incluye:
- link privado,
- resumen del trabajo,
- fotos grandes,
- checklist,
- firma cliente,
- confirmacion servicio completado.

Resultado:
El cliente puede ver y confirmar sin cuenta.

## Fase 5: Reportes y estimaciones

Incluye:
- horas por dia/semana/mes,
- ingresos estimados,
- tarifas por hora/fijo,
- IVA/descuentos configurables como estimacion,
- exporte simple CSV/PDF si es viable.

Resultado:
El independiente ya siente valor financiero.

## Fase 6: Empresa basica

Incluye:
- crear empresa,
- cleaners,
- clientes,
- trabajos,
- asignacion de cleaner,
- calendario empresa,
- dashboard empresa.

Resultado:
Una empresa pequena puede operar trabajos con varios cleaners.

## Fase 7: Empresa Pro

Incluye:
- pagos externos registrados,
- firma del cleaner por pago recibido,
- reportes por cleaner,
- pagos pendientes,
- firmas pendientes,
- exportes.

Resultado:
La empresa tiene control operativo y comprobantes internos.

## Fase 8: Notificaciones

MVP:
- email.

Luego:
- WhatsApp,
- SMS,
- push.

Eventos:
- recordatorio de manana,
- trabajo asignado,
- cleaner llego,
- trabajo terminado,
- firma pendiente,
- resumen listo.

## Fase 9: Pulido premium

- mejores dashboards,
- calendario visual,
- mapas,
- filtros,
- plantillas de checklist,
- branding por empresa,
- portal cliente mas bonito,
- multidioma completo.

## Lo que NO entra en MVP inicial

- procesar pagos reales,
- contabilidad legal oficial,
- app nativa,
- GPS continuo,
- bancos,
- automatizacion fiscal mundial,
- ruta optimizada avanzada.

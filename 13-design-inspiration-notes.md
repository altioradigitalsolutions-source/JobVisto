# Design Inspiration Notes

Busqueda realizada sobre SaaS de field service, limpieza y operaciones.

## Patrones importantes

### ZenMaid

Enfatiza:
- scheduling para limpieza,
- automatizacion,
- comunicaciones con clientes,
- mobile app para cleaners,
- vistas flexibles: dispatch, lista, mapa.

Leccion:
La agenda/calendario debe ser protagonista, no una funcion secundaria.

### iTask

Enfatiza:
- admin portal,
- mobile app,
- limpieza en tiempo real,
- roles/permisos,
- GPS o QR,
- checklist y photo verification.

Leccion:
Separar claramente escritorio/admin y movil/field staff.

### SaberTask

Enfatiza:
- plataforma completa,
- dashboard,
- app movil nativa,
- GPS clock in/out,
- stats personales,
- photo documentation,
- daily task list.

Leccion:
El cleaner debe sentir que tiene su propia app, no una version reducida del admin.

### ScheduleCTRL

Enfatiza:
- modern command centre,
- dashboard,
- online booking,
- realtime scheduling,
- providers,
- timelogs,
- invoices,
- payroll,
- customer communications.

Leccion:
El mensaje comercial fuerte es "centro de comando moderno".

### Jobber / Housecall Pro

Enfatizan:
- dashboard,
- acciones recomendadas,
- asignaciones,
- proximos trabajos,
- balances,
- app movil para field teams.

Leccion:
El dashboard debe responder "que tengo que hacer ahora".

## Direccion visual para JobVisto

La web debe mostrar:
- Hero con producto visible, no solo imagen de limpieza.
- Dashboard realista con calendario y trabajos.
- Mobile cleaner app realista.
- Cliente link preview.
- Fotos grandes de evidencia.
- Reportes/contabilidad estimada.

La UI debe verse como un sistema por el que alguien pagaria:
- espaciado fino,
- estados claros,
- tablas limpias,
- calendario visual,
- colores profesionales,
- botones con jerarquia,
- datos reales de ejemplo.
